# Codex Review Prompt

## 種類
これは **要件定義書（plan）レビュー** です。コードレビューではありません。

## 使い方
- Claude Code 内で `/plan-review` を入力すると、このファイルを使って Codex にレビューを投げる

## レビュー対象
以下に **絶対パス**が渡されます。そのファイルだけを読んでレビューしてください。
作業担当の割当や worker_XXX.yaml の参照は不要です。

以下の計画書をレビューしてください。

## 観点
- 抜け漏れ
- 妥当性（要件/仕様/影響）
- リスクや懸念
- 実行可能性（タスク分解・テスト・移行）

## 出力形式
1. 問題点・不足点
2. 追加で確認したい質問
3. 改善案

問題がなければ「問題なし」と明記してください。
